Dicionário de Tags HTML

Dicionário de Tags Utilizadas

  • <html> - Define a estrutura do documento HTML.
  • <head> - Contém metadados, links para arquivos externos e o título da página.
  • <meta> - Define informações sobre a página, como o charset.
  • <link> - Vincula um arquivo externo, como CSS.
  • <header> - Representa o cabeçalho de uma página.
  • <hr> - Insere uma linha horizontal (divisor de conteúdo).
  • <div> - Container genérico para agrupar conteúdo
  • <nav> - Representa uma seção de navegação.
  • <body> - Contém o conteúdo visível da página.
  • <section> - Define uma seção de conteúdo temático.
  • <strong> - Aplica negrito ao texto.
  • <h1> - <h6> - Tags de cabeçalho.
  • <p> - Define um parágrafo de texto.
  • <a> - Cria um link.
  • <img> - Exibe uma imagem.
  • <ul> - Define uma lista não ordenada.
  • <li> - Define um item de lista.
  • <footer> - Define o rodapé da página.
  • <address> - Define informações de contato.
